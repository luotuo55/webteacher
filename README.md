# H5 课件托管系统

本项目托管了以下精品互动课件：

1. **9加几 - 趣味数学课** (`/lessons/ninepluse/`)
2. **数学排队问题** (`/lessons/diji/`)
3. **8、7、6加几** (`/lessons/nineplus_v2/`)

## 启动开发环境

```bash
# 安装依赖
npm install --legacy-peer-deps

# 启动开发服务器
npm run dev
```

访问：http://localhost:3000

## 构建与部署

```bash
npm run build
```

构建结果位于 `dist/` 目录。
