# 课件预览说明

## 快速预览步骤

### 第一步：确保图片文件已复制
请确保 `public/主题图.png` 文件存在。如果还没有，请：
- 双击运行 `copy-images.bat` 文件，或
- 手动将 `主题图.png` 复制到 `public/` 目录

### 第二步：安装依赖
在项目根目录打开终端（PowerShell 或 CMD），运行：

```bash
npm install
```

这将安装 Vue 3 和 Vite 等必要的依赖包。

### 第三步：启动开发服务器
运行以下命令：

```bash
npm run dev
```

### 第四步：在浏览器中打开
命令运行后，终端会显示类似以下的信息：

```
  VITE v4.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

**在浏览器中打开显示的地址**（通常是 `http://localhost:5173/`）即可预览课件。

## 预览功能

在开发模式下，你可以：
- ✅ 实时查看课件效果
- ✅ 修改代码后自动刷新（热更新）
- ✅ 测试所有交互功能
- ✅ 查看控制台错误信息（如有）

## 其他命令

### 构建生产版本
```bash
npm run build
```
构建完成后，文件会输出到 `dist/` 目录，可以直接部署。

### 预览构建后的文件
```bash
npm run preview
```
预览构建后的生产版本。

## 常见问题

### 1. 提示 "npm 不是内部或外部命令"
- 需要先安装 Node.js：https://nodejs.org/
- 安装完成后重新打开终端

### 2. 图片不显示
- 检查 `public/主题图.png` 文件是否存在
- 如果不存在，运行 `copy-images.bat` 复制图片

### 3. 端口被占用
- Vite 会自动尝试其他端口
- 或者手动指定端口：`npm run dev -- --port 3000`

### 4. 页面空白
- 打开浏览器开发者工具（F12）查看控制台错误
- 检查是否有 JavaScript 错误

## 浏览器要求

建议使用现代浏览器：
- Chrome 90+
- Firefox 88+
- Edge 90+
- Safari 14+
